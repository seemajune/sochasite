### Set Local Environment to Devel
export NODE_ENV=development

### Local Server
run local (localhost:9000)
`nodemon`

### Local Database
run local mongo db (from mongodb/bin/)
./mongod --dbpath ../../soshasite/data

./mongo util
use sochasite
